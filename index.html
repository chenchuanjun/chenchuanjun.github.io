ccs3的3d变换和动画特效。

坐标分为三个轴：X轴，Y轴和Z轴

1. X轴表示在水平方向，正值表示水平向右。
2. Y轴表示在垂直方向，正值表示垂直向下。
3. Z轴表示垂直于屏幕方向，正值朝向屏幕外。

css3的旋转效果实现。

左手原则：大拇指指向轴的朝向，四指就是旋转的方向。
例如： rotateX(45deg),rotateY(45deg)或者rotateZ(45deg);
浏览器对transform 的解析是从左向右的。

近大远小效果的实现。

一般给父元素或着祖先元素设置perspective属性来实现。

实质上是表示元素距离显示屏幕的大小。



开启浏览器的3D变换效果。

1.为3变换的父元素设置此属性 transform-style: preserve-3d
2.父元素如果设置了opacity 属性，那么渲染效果不会起作用。

动画效果的设置

  • 1.首先定义动画
  • 2.然后在设置动画的属性
    • animation-name: 动画的名称。
    • animation-duration: 动画的持续时间。
    • animation-timing-function:动画的线性 linear.
    • animation-iteration-count:动画的播放次数 infinate
    • animation-play-state: 动画的播放次数 running/paused.

      定义动画的播放阶段。


      如果在上一个阶段中设定了一个动画属性后，下一个阶段不设定次属性的话，会还原成默认值
  • 动画结束时保持设置状态的属性设定。
    • animation-fill-mode:forwards;

      动画属性的合写。

      • 第一次出现的是持续时间，第二次出现的是延迟时间，其他值没有顺序要求
      • 有多个动画属性合写时用逗号分割。

        动画库的使用。

  • 引入动画外链
  • 设置类名。

弹性布局的基本使用。

  • 设置了 display: flex 属性之后 称 盒子 为弹性盒子，子元素 就应用了 弹性布局
  • 弹性布局开启后，默认分为两个轴：主轴和副轴。
  • 主轴 ：从左往右 x方向
    • 设置主轴的对齐方式: justify-content:flex-start;
    1. lex-end;主轴的末尾
    2. flex-start:主轴的 开始
    3. space-around 会预留一些间隙
    4. space-between 元素之间 会预留间隙 但是 两边顶格。
  • 设置副轴的对其方式: align-items: flex-start;
    1. flex-end; 底部
      2.center 中间
      3.flex-start 默认值;
  • 设置子项单独在侧轴（纵轴）方向上的对齐方式: align-self
    1. align-self: flex-end.
    2. align-self: flex-start
    3. align-self:center;

#1.css3中的属性选择器

1.1属性选择器的分类

1.    对象[属性]    包含此属性的元素。
2.    对象[属性 = "a"]    属性值里为a的元素。
3.    对象[属性 ~=  "a"]    属性值用空格进行分割，有一个为a.
4.    对象[属性^ = "a"]    属性值以a开始。
5.    对象[属性$ = "a"]    属性值以a结束。
6.    对象[属性* = "a"]     属性里包含有值a,可以是字母。

#2.兄弟选择器
1.1 写法 选择器1~选择器2
1.2 只能获取在相同父元素，获取选择器1以后，与选择器2类型相同的元素。

#3.伪类选择器

1.    :first-child    第一个元素。
2.    ：last-child        最后一个元素。
3.    ：nth-child(2)    索引值从1开始，还可以传入表达式。
        >:nth-child(2n)    n是一个依次累加的值
        >：nth-child(2n-1) n从零开始计数，n不能写到后面。
4.：nth-last-child(3)    从后往前找。
5. 若不考虑标签的类型， 推荐使用：nth-child();
6. 若考虑类型， 或多个标签混合时，使用 ：nth-of-type(1).
7. :first-of-type{}  该类型的第一个
8. :last-of-type{}
