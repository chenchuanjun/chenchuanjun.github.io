#1.css3中的属性选择器

1.1属性选择器的分类

1.    对象[属性]    包含此属性的元素。
2.    对象[属性 = "a"]    属性值里为a的元素。
3.    对象[属性 ~=  "a"]    属性值用空格进行分割，有一个为a.
4.    对象[属性^ = "a"]    属性值以a开始。
5.    对象[属性$ = "a"]    属性值以a结束。
6.    对象[属性* = "a"]     属性里包含有值a,可以是字母。

#2.兄弟选择器
1.1 写法 选择器1~选择器2
1.2 只能获取在相同父元素，获取选择器1以后，与选择器2类型相同的元素。

#3.伪类选择器

1.    :first-child    第一个元素。
2.    ：last-child        最后一个元素。
3.    ：nth-child(2)    索引值从1开始，还可以传入表达式。
        >:nth-child(2n)    n是一个依次累加的值
        >：nth-child(2n-1) n从零开始计数，n不能写到后面。
4.：nth-last-child(3)    从后往前找。
5. 若不考虑标签的类型， 推荐使用：nth-child();
6. 若考虑类型， 或多个标签混合时，使用 ：nth-of-type(1).
7. :first-of-type{}  该类型的第一个
8. :last-of-type{}

3.    ::palceholder    在使用时需要添加前缀。

4.    ::selection    设置对象的被选择的样式：color background-color text-shadow(文字阴影).
5.    ::before与::after

#6.text-shadow

##6.1字体阴影
>    基本写法 text-shadow(x轴偏移值 y轴偏移值 颜色)
>    第四个属性    text-shadow(x轴偏移值 y轴偏移值 阴影的模糊程度 颜色)
>    如果要设置多个阴影，要通过逗号来分割不同的阴影。
##6.2盒子阴影
>    属性和写法和字体阴影类似，但含有 inset可以设置向内发散。
